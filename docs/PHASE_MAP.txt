=======================================================
技術要件サマリー
=======================================================

コア設定:
- MCP エンドポイント: 127.0.0.1:39200 (固定)
- 設定ファイル: mcp.json (Read-Only モード)
- 書込みゲート: UCOMM_ENABLE_WRITES環境変数制御
- セキュアモード: UCOMM_SECURE_MODE=1 時は書込み全面禁止

パフォーマンス目標:
- 初回実行レイテンシ: ≤ 6s
- CI 実行レイテンシ: ≤ 6s
- MCP 通信レイテンシ: ≤ 500ms
- 成功率: Phase別に段階的向上 (35% → 50% → 70% → 95%)

プラットフォーム:
- サポート OS: Ubuntu 20.04+, Windows Server 2019+, macOS 12+
- 実行環境: GitHub Actions (Ubuntu/Windows)
- 開発環境: tmux ベース組織化

品質保証:
- 自動テスト: smoke.yml による CI/CD
- 三分診システム: scripts/ci_triage.sh による障害分析
- 証跡管理: artifacts/ ディレクトリ構造化
- レポート体系: docs/reports/ 統一フォーマット
